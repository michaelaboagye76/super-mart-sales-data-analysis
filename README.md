# Retail Store Sales Data Analysis (Excel)

## Project Overview
This project analyzes a sales dataset using **Microsoft Excel** to uncover trends, measure performance against targets, and identify opportunities for cost savings and profitability improvements.  

The dataset includes information such as *orders, customers, discounts, shipping details, and product categories*.  
Through analysis and visualization, the project demonstrates how data-driven insights can support better business decisions.  

## Dataset Columns
- **Order Details**: Row ID, Order ID, Order Date, Order Quantity, Order Priority  
- **Financials**: Sales, Sales Target, Discount, Profit, Unit Price, Discounted Price  
- **Logistics**: Ship Mode, Shipping Cost, Ship Date  
- **Customer Info**: Customer Name, Zipcode, City, State, Region, Customer Segment  
- **Product Details**: Product Category, Sub-Category, Product Name, Supplier, Product Container  
- **Marketing & Margin**: Marketing Channel, Product Base Margin  

## Tools Used
- **Microsoft Excel**: Data cleaning, analysis, pivot tables, and dashboards , Charts, KPIs, and trend analysis

  

## Business Impact
*This project demonstrates how Excel-based analytics can:*  

- Improve profitability through pricing and margin analysis  
- Reduce operational costs by optimizing shipping strategies  
- Enable smarter marketing through customer segmentation  
- Provide decision-makers with data-driven insights
